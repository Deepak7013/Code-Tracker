# Code Tracker Chrome Extension

Code Tracker helps programmers keep track of problems solved across platforms like LeetCode, CodeChef, HackerRank, and HackerEarth.  
It scans the active tab to save the problem title, difficulty, code, and URL for quick access later.

---

## Installation

Follow these steps to install the extension locally:

1. Download the project from GitHub.
2. Click the **Code** button on the repository page.
3. Select **Download ZIP**.
4. Extract the downloaded ZIP file.
5. Open **Google Chrome** and go to:

   chrome://extensions

6. Enable **Developer Mode** (top-right corner).
7. Click **Load Unpacked**.
8. Select the **code-tracker** folder that contains `manifest.json`.

The extension will now appear in your Chrome extensions list.

---

## Features
- Automatically capture problem title
- Save difficulty level
- Store the code you wrote
- Save the problem URL
- Quick access to previously solved problems
